## Cellular-Automata
 
### Setup: 

1. Open terminal in this directory
2. Enter virtual environment: 
<br>`source venv-dir/bin/activate`
3. Install `pygame` library in virtual env:
<br>`pip3 install pygame`

### Run:

1. Open terminal in this directory
2. Enter virtual environment:
<br>`source venv-dir/bin/activate`
3. Start:
<br>`python3 src/main.py`
